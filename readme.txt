# Devine Le Nombre

## Description

Bienvenue dans "Devine Le Nombre" ! C'est un jeu simple où tu dois deviner un chiffre aléatoire entre 1 et 100 (ou plus selon la difficulté choisie) 
en un nombre d'essais illimité. 
Le jeu t'indiquera si ton chiffre est trop grand ou trop petit et te donnera un indice sonore à chaque mauvaise tentative.
 Une fois que tu as trouvé le bon chiffre, le jeu te félicite et affiche le temps que tu as mis pour le deviner.

Tu peux également choisir la difficulté avant de commencer, avec trois niveaux différents :

- **Facile** : un nombre entre 1 et 50
- **Normal** : un nombre entre 1 et 100
- **Difficile** : un nombre entre 1 et 500

Le jeu suit également ton meilleur score (le plus faible nombre d'essais) et le temps qu'il t'a fallu pour deviner le nombre !

## Fonctionnalités

- **Difficulté au choix** : Facile, Normal, Difficile.
- **Sons d'indication** : Son de mauvaise réponse et son de victoire.
- **Chronomètre** : Le temps que tu mets pour deviner le nombre est enregistré.
- **Réessayer** : Tu peux jouer plusieurs fois et améliorer ton score.

## Prérequis

Ce projet nécessite Python 3.x pour être exécuté correctement. De plus, il utilise le module `winsound` pour les effets sonores, disponible uniquement sous Windows.

### Dépendances

- **Python 3.x** : Télécharge et installe Python depuis [python.org](https://www.python.org/downloads/)
- **Module `winsound`** : Ce module est intégré à Python sous Windows, il n'y a donc pas besoin d'installer quoi que ce soit.
